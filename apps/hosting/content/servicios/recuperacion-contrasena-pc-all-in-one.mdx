## Definición
El bloqueo de una All-in-One puede ocurrir por el olvido de credenciales de usuario, bloqueos de cuenta Microsoft/iCloud tras una actualización, o restricciones a nivel de hardware (Password de BIOS). Este servicio utiliza herramientas de bajo nivel para editar los registros de seguridad y permitir el ingreso al escritorio, garantizando que la información del disco duro permanezca intacta.

### Diagnóstico de Acceso
* **Identificación del Bloqueo**: Determinación de si el impedimento es a nivel de Sistema Operativo (Windows/macOS) o de Firmware (BIOS/UEFI).
* **Verificación de Cifrado**: Comprobación del estado de BitLocker o FileVault para evaluar la viabilidad de la recuperación sin llave de recuperación.
* **Análisis de Privilegios**: Evaluación de cuentas de administrador vs. cuentas estándar para restaurar el control total del equipo.
* **Validación de Identidad**: Protocolo de seguridad obligatorio para confirmar la propiedad del equipo antes de proceder.

## Procedimiento
1. **Reset de Password (OS)**: Bypass o restablecimiento de la base de datos SAM (Windows) o restablecimiento de llavero (Mac) para generar una nueva credencial de acceso.
2. **Desbloqueo de BIOS/UEFI**: Remoción de contraseñas de arranque o de configuración mediante procesos electrónicos o de software especializado.
3. **Optimización de Seguridad**: Configuración de métodos de recuperación alternativos (preguntas de seguridad o vinculación de correo) para evitar futuros incidentes.

## Especificaciones
| Nivel de Bloqueo | Método de Intervención | Impacto en Datos |
| :--- | :--- | :--- |
| Usuario Local | Edición de Registro | Cero pérdida de datos |
| Cuenta en la Nube | Recuperación de Tokens | Conservación de perfiles |
| BIOS / UEFI | Reset de Hardware / Firmware | Acceso total a configuración |

## Garantía
Nuestra garantía técnica de 90 días asegura que el nuevo método de acceso sea estable y que no existan conflictos con las actualizaciones de seguridad del sistema. El servicio concluye una vez que el cliente verifica el acceso total a su escritorio y documentos.

## Límites
* **Cifrado de Alto Nivel**: Equipos con BitLocker activo o chip de seguridad T2 (Mac) sin clave de recuperación podrían requerir un formateo si la llave es inaccesible.
* **Políticas Legales**: Nos reservamos el derecho de rechazar el servicio si no se acredita la propiedad legítima del dispositivo.

## Resultados
* Acceso inmediato a todos tus documentos, fotos y aplicaciones.
* Eliminación de barreras de inicio de sesión que impiden el trabajo.
* Configuración de un entorno de acceso más seguro y recuperable.
